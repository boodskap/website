Legends

Data Points Data point is a specific piece of information collected from IoT devices, such as sensor readings, device status, or user interactions, which is used for analysis and decision-making.
Device Library Device library describes the architecture and functionality of a connected device, detailing how it communicates, what are all the parameters (data) it communicates, etc.
Device Parameter Device parameter is a measurable attribute or setting of an IoT device, such as temperature, humidity, or battery level, that can be monitored or controlled remotely.
Dashboard Dashboard is a user interface that provides real-time monitoring, control, and data visualization of connected devices and systems
Device Device is a physical object embedded with sensors, software, and connectivity to exchange data with other devices and systems over the internet
User User is an individual or entity that utilizes software systems to manage, monitor, and control IoT devices and data.
Client Client account allows users or devices under a separate account to access, manage, and interact with IoT services and data, often with specific permissions and roles
Archival Data archival refers to the process of systematically deleting device data from to optimize storage and maintain performance.
